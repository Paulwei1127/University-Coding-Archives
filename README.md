# University Coding Archives

This repository serves as an archive for my coursework and small-scale projects developed during my university years. It documents my learning process across various domains, including Assembly Language, Object-Oriented Programming, and Frontend Basics.

收錄了一些我在大學時期的課程作業與小型練習專案。雖然這些專案規模較小，但它們見證了我在不同技術領域的學習歷程與基礎實作能力。

##收錄專案 (Project List)

### 1. RISC-V Recursive Function
* **Path:** `./RISC-V_Recursive_Function/`
* **Language:** RISC-V Assembly
* **Highlight:**
    * Implemented a recursive mathematical function $F(x)$ using assembly.
    * Manually managed Stack Pointer (`sp`) and Return Address (`ra`) to handle nested function calls.
    * 實作遞迴函式，展示對記憶體堆疊與函式呼叫慣例 (Calling Convention) 的理解。

### 2. Java Strategy Board Game (攻城掠地)
* **Path:** `./Java_Strategy_Board_Game/`
* **Language:** Java (Swing/AWT)
* **Highlight:**
    * A 2-player strategy board game with a graphical user interface.
    * Implemented object-oriented logic for piece movement, capturing rules, and win condition checks.
    * 包含圖形介面、棋子移動判定與勝負邏輯的雙人對戰遊戲。

### 3. Web Roulette Game (輪盤遊戲)
* **Path:** `./Web_Roulette_Game/`
* **Language:** HTML5, CSS, JavaScript
* **Highlight:**
    * An interactive roulette game using HTML5 Canvas for rendering animations.
    * Utilized trigonometry (`Math.cos`, `Math.sin`) to calculate wheel rotation and ball position.
    * 使用 Canvas 繪製旋轉動畫，並包含下注與賠率計算邏輯。

### 4. Sierpinski Triangle (謝爾賓斯基三角形)
* **Path:** `./Sierpinski_Triangle/`
* **Language:**  HTML5 / WebGL2 (JavaScript)
* **Highlight:**
    *A fractal rendering program using WebGL2 API for hardware acceleration.
	*Implemented Recursive Subdivision algorithm to generate the triangle pattern.
	*Features an Interactive UI to adjust recursion depth (layers), position, rotation, and color modes in real-time.
	*使用 WebGL2 進行圖形渲染，實作遞迴分割演算法，並提供即時調整層數與顏色的互動介面。
